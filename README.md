Installation
--------------------
1. Clone
git clone https://github.com/Indranil-Sarmah/Portfolio.git

2. Navigate to the project directory
3. Install dependencies:
npm install

4. Run it by
npm run dev

5. Customize it according to your requirements. 
6. Thank me by giving a star in the repo.
